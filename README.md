# todo-app
## General info
It as a training project to work on MySQL Database

## Technologies
> Project is created with:
- [x] node.js
- [x] express.js
- [x] MySQL
- [x] Sequelize ORM
- [x] ejs (template engine)
- [x] Bootstrap 5
- [x] font-awesome
*
> the project is made with MVC design pattern

## Setup
To run this project, install it locally using npm:
- $ npm install
- $ npm start
