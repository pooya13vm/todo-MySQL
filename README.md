# todo-app
## General info
It as a training project to work on MySQL Database

## Technologies
#### Project is created with:
#### node.js
#### express.js
#### MySQL
#### Sequelize ORM
#### ejs (template engine)
#### Bootstrap 5
#### font-awesome
*
#### the project is made with MVC design pattern

## Setup
#### To run this project, install it locally using npm:
#### $ npm install
#### $ npm start
